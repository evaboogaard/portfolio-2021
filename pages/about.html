I was born on the 19th of June, 1999. I have always loved using different mediums to create art. Drawing, painting, sketching, digital art, physical art, I’ve tried it all. Eventually, my heart has settled on using HTML, CSS and Javascript to create visual designs.

The drivenness I experience within web design isn’t rooted in the way the web is designed at this very moment, but in the way it could be in the future. I am especially interested in the accessibility of the web.
